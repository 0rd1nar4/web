У цій папці зібрані матеріали з дисципліни "Веб-програмування та веб-дизайн".
Структура папки:
1. project
   Це основний проєкт курсової роботи.
   Тема: інформаційний портал "Найактуальніші спортивні події університету".
   У проєкті використано:
   - HTML5;
   - CSS3;
   - JavaScript;
   - Flexbox;
   - CSS Grid;
   - Media Queries.
   У папці project є сторінки:
   - index.html - головна сторінка;
   - about.html - сторінка "Про сайт";
   - news.html - сторінка новин;
   - gallery.html - галерея;
   - contacts.html - контакти.
   Для запуску потрібно відкрити файл:
   project/index.html
   Сайт працює локально у браузері. Сервер для запуску не потрібний.
   Для карти Google на сторінці контактів потрібне підключення до інтернету.
2. labs
   Це папка з лабораторними роботами.
   Кожна лабораторна робота розміщена в окремій папці:
   - lab1_html;
   - lab2_html;
   - lab3_html;
   - lab4_html;
   - lab5_html;
   - lab6_html;
   - lab7_html.
   У кожній лабораторній є файл index.html.
   Для запуску потрібно відкрити index.html у відповідній папці.
   У лабораторних роботах використано простий HTML, CSS і JavaScript.
   Код написаний у навчальному стилі без фреймворків і складної архітектури.
3. kursova.docx
   Це текст курсової роботи у форматі Word.
   У документі описано мету роботи, структуру сайту, використані технології,
   реалізацію сторінок, JavaScript-логіку, адаптивність і висновки.
